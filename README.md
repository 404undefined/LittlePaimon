# 使用方法

> 环境准备：一台电脑或云服务器，Windows或Linux都可。
>
> Python3.8及以上，安装ffmpeg。
1. 克隆\下载项目
  * 使用git（推荐，方便版本更新）
    - 安装[git](https://git-scm.com/)
    
    - 命令行cd到存放目录，进行克隆
    
      ```
      git clone https://github.com/CMHopeSunshine/LittlePaimon
      ```
      

  * 直接下载，解压
2. 安装依赖
      ```
      cd LittlePaimon # 进入LittlePaimon目录
      pip install -r requirements.txt
      ```
