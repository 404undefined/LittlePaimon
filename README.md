原神多功能机器人，通过米游社接口查询uid的游戏信息，并提供WIKI查询和各种各样的好玩的功能。

本分支为将派蒙功能拆分成独立的Nonebot2插件所用，请勿直接克隆本分支内容，派蒙主要分支在：
